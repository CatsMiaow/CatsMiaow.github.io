역할

기본 역할#

서버에 뉴비(새내기)가 입장할 때 지정한 역할을 부여합니다.

  • !역할 기본 역할명
    • 봇이 역할 관리 권한을 가지고 있어야 하고, 봇이 가진 역할보다 낮은 역할만 설정할 수 있습니다.
    • 기본 역할을 @everyone이 아닌 다른 역할을 부여하여 역할 이름/색상/설정을 변경할 수 있습니다.
  • !역할 기본 삭제
    • 지정된 기본 역할을 삭제합니다.

부여 역할#

다른 유저에게 부여할 수 있는 역할을 설정합니다.

  • !역할 부여 역할명
    • 봇보다 낮은 역할만 설정할 수 있습니다.
    • 특정 역할을 지정하여 특정 유저가 다른 유저에게 수동으로 역할을 부여할 때 사용할 수 있습니다.
  • !역할 부여 @멘션
    • 역할 부여를 설정한 후에 사용할 수 있습니다.
    • 봇보다 높은 역할의 유저만 사용할 수 있습니다.
  • !역할 부여 삭제
    • 설정된 역할 부여를 삭제합니다.

요청 역할#

유저가 스스로 역할을 가져갈 수 있도록 설정합니다.

  • !역할 요청 교체
    • 봇이 가진 역할보다 낮은 역할만 가져갈 수 있습니다.
    • 기존 역할을 요청하는 역할로 교체합니다.
    • 역할을 교체하기 때문에 기존 역할이 봇보다 높으면 수행할 수 없습니다.
  • !역할 요청 추가
    • 역할을 교체하지 않고 기존 역할에 역할을 추가하도록 설정합니다.
    • 나머지는 위의 교체 설정과 같습니다.
  • !역할 요청 삭제
    • 설정된 역할 요청을 삭제합니다.
  • !역할 요청 역할명
    • 역할 요청을 설정한 후에 사용할 수 있습니다.
    • 서버에 처음 입장한 유저가 진영(?)을 선택할 때 사용할 수 있습니다.
  • !역할 삭제 역할명
    • 요청한 역할을 삭제할 수 있습니다.
    • 요청과 같이 봇이 가진 역할보다 낮은 역할만 삭제할 수 있습니다.
