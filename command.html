명령어 목록

알림

모든 명령어는 디스코드의 슬래시(/) 명령어로 사용할 수 있습니다.
일부 명령어는 편의성을 위해 텍스트(!) 명령어를 지원합니다.
슬래시 명령어 제한은 ⚙️서버 설정 > 연동 > 봇과 앱 > 마냥 클릭(관리) > 명령어에서 제한할 수 있습니다.

  • 기본 명령어
    • /명령어 - 봇 명령어 안내 링크를 보여줍니다.
    • /초대 - 다른 서버에 봇을 초대할 수 있는 링크를 띄워줍니다.
    • /핑 - 봇의 최근 응답 속도(ms)를 알려줍니다.
  • 서버 관리
    • /내서버 - 서버의 봇 설정을 보여주거나 알림을 받을 관리자를 등록합니다.
    • /공지 - 서버에 뉴비(새내기)가 입장할 때 보여줄 메시지를 등록합니다.
    • /역할 - 역할을 요청하거나 부여합니다.
    • /스팸 - 스팸 메시지를 보낸 유저를 제재합니다.
    • /로그 - 유저 활동 로그를 메시지로 기록합니다.
    • /청소 - 입력한 개수 범위 내의 채팅 메시지를 삭제합니다.
    • /뮤트 - 서버에서 유저의 채팅을 금지합니다.
    • /킥 - 서버에서 유저를 추방합니다.
    • /슬로우 - 채널의 슬로우 모드를 설정합니다.
    • /잠금 - 채널의 메시지 보내기를 잠급니다.
  • 게임
    • /강화 - 아이템을 강화합니다.
    • /주식 - 실제 종목으로 주식 게임을 합니다.
    • /한겜 - 한글 게임을 시작합니다.
    • /타자 - 타자 게임을 시작합니다.
    • /러시안 - 러시안 룰렛을 시작합니다.
    • /경마 - 이모지를 사용하여 경주합니다.
    • /배라 - 배스킨라빈스31 게임을 시작합니다.
    • /야구 - 숫자야구 게임을 시작합니다.
    • /반응 - 반응속도 게임을 시작합니다.
  • 재밋거리
    • /이미지 - 이미지 검색 결과에서 무작위로 하나를 올립니다.
    • /출석 - 출석 체크를 설정합니다.
    • /투표 - 이모지로 항목별 투표를 시작합니다.
    • /로또 - 로또 번호를 생성하고 추첨합니다.
    • /골라 - 입력한 항목 중에서 무작위로 골라줍니다.
    • /주사위 - 범위 내의 무작위 숫자를 골라줍니다.
    • /운세 - 띠별 운세 정보를 보여줍니다.
  • 유틸리티
    • /계산 - 입력된 수식을 계산합니다, 다양한 수식이 가능합니다.
    • /알람 - 설정한 시간에 개인 메시지(DM)로 알람을 보냅니다.
    • /번역 - 한국어/영어/일어/중국어를 번역합니다.
    • /로마자 - 한글 이름을 로마자 표기로 보여줍니다.
    • /시간 - 해당하는 국가의 현재 시간을 보여줍니다.
  • 정보 검색
    • /유튜브 - 유튜브 검색 결과를 보여줍니다.
    • /날씨 - 해당하는 지역의 날씨 정보를 보여줍니다.
    • /환율 - 국가별 환율 정보를 보여줍니다.
    • /검색 - 구글 검색 결과를 보여줍니다.
    • /국어 - 국어사전 검색 결과를 보여줍니다.
    • /영어 - 영어사전 검색 결과를 보여줍니다.
    • /사전 - 백과사전 검색 결과를 보여줍니다.
    • /뉴스 - 뉴스 검색 결과를 보여줍니다.
    • /영화 - 영화 검색 결과에서 최근작을 보여줍니다.
    • /쇼핑 - 쇼핑 검색 결과에서 유사한 상품 하나를 보여줍니다.
    • /업체 - 업체 검색 결과에서 유사한 상호 하나를 보여줍니다.
    • /도서 - 도서 검색 결과에서 유사한 도서 하나를 보여줍니다.
    • /비트 - 가상화폐 시세 정보를 보여줍니다.
    • /나무 - 검색어에 해당하는 나무위키 링크를 보여줍니다.
    • /스팀 - 스팀 게임 정보와 접속자 순위를 보여줍니다.
    • /트위치 - 트위치 시청자 순위를 보여줍니다.
    • /애니 - 애니메이션 정보를 보여줍니다.
    • /실검 - 실시간 인기 검색어 순위를 보여줍니다.
  • 정보 관리
    • /내정보 - 자신의 디스코드 정보를 보여줍니다.
    • /토픽 - 현재 채널의 채널 주제를 보여줍니다.
    • /메모 - 단어에 대한 설명을 메모합니다.
  • 구독자
    • /구독설정 - 구독자 전용 설정값을 입력합니다.
    • /알람 - 멀티 알람을 설정합니다.
    • /말 - 입력한 문장을 TTS로 재생합니다.
    • /뮤 - 유튜브 검색으로 음악을 재생합니다.
  • 지원 서버
    • /도움 - 봇 이용 안내 가이드를 보여줍니다.
    • /상태 - 서버 아이디로 봇 상태를 확인합니다.
